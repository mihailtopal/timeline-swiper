# Timeline Swiper

Интерактивный таймлайн с анимированными событиями, Swiper-слайдером и круговой навигацией по периодам.

##  Стек

- React
- TypeScript
- Swiper
- SCSS (CSS Modules)
- Webpack

##  Установка

```bash
git clone  https://github.com/mihailtopal/timeline-swipper.git

cd timeline-swiper

npm install
```

##  Запуск проекта

```bash

npm run dev
```
